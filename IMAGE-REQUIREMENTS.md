## Image Guidelines

### Specifications:
- **Format:** JPG or WebP (WebP preferred for smaller file sizes)
- **Max file size:** 2MB per image
- **Resolution:** 
  - Hero images: 1920x1080px minimum
  - Gallery images: 1200x800px minimum
  - Thumbnails: Auto-generated by Astro

### Photography Tips:
- ✅ Use natural lighting when possible
- ✅ Shoot in landscape orientation for most photos
- ✅ Capture key features: lake views, sunsets, interiors
- ✅ Include lifestyle shots (people enjoying the space - optional)
- ✅ Take photos during golden hour for best lighting
- ✅ Ensure rooms are clean and staged

### Image Optimization:
Before uploading, consider compressing your images:
- Use tools like TinyPNG, Squoosh, or ImageOptim
- Aim for balance between quality and file size
- Astro will further optimize images during build

## How to Add Images

1. **Place images** in the appropriate `/public/images/` subfolder
2. **Update references** in content files if filenames differ
3. **Test locally** by running `npm run dev`
4. **Commit and deploy** your changes
