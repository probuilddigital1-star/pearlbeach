## SEO Checklist for New Content

When adding new pages or content:

- [ ] **Title:** 50-60 characters, includes primary keyword
- [ ] **Description:** 150-160 characters, compelling with CTA
- [ ] **Keywords:** 5-10 relevant keywords, comma-separated
- [ ] **H1:** Only one per page, includes primary keyword
- [ ] **H2/H3:** Logical hierarchy, includes secondary keywords
- [ ] **Breadcrumbs:** Implemented on all pages except homepage
- [ ] **Images:** Alt text with descriptive keywords
- [ ] **Internal Links:** Link to relevant pages
- [ ] **Schema:** Add if applicable (property, review, FAQ)

## Schema Markup Quick Guide

### When to Use Each Schema Type:

**LocalBusiness** (Homepage only)
- Business overview
- Contains: name, address, phone, hours, rating
- Use once per site

**VacationRental** (Individual cottage pages)
- Specific property details
- Contains: rooms, amenities, capacity, price
- Use once per property page

**Review** (Reviews page)
- Guest testimonials
- Contains: author, date, review text
- Can have multiple reviews

**BreadcrumbList** (All pages via component)
- Navigation structure
- Automatically generated by Breadcrumbs component

**FAQPage** (FAQ page only)
- Question/answer pairs
- Contains: question text, answer text
- Eligible for rich snippets

## Emergency SEO Issues

### Issue: Rankings Dropped Suddenly
**Check:**
1. Search Console for manual penalties
2. Site is accessible (not down)
3. Robots.txt not blocking pages
4. Sitemap is accessible
5. Google algorithm update

### Issue: Rich Snippets Not Showing
**Remember:**
- Google chooses snippets algorithmically
- Not guaranteed even with valid schema
- Can take 4-6 weeks to appear
- Check validation with Rich Results Test

### Issue: Pages Not Indexed
**Check:**
1. Sitemap includes the page
2. Robots.txt allows crawling
3. Page is linked from other pages
4. Submit URL in Search Console
5. Wait 1-2 weeks for indexing
